# UI/UX Guidelines

## 1. Design System

### 1.1 Typography
1. **Font Families**
   - Primary: Geist Sans
   - Monospace: Geist Mono
   - Fallback: System fonts

2. **Type Scale**
   - Display: 48px/60px
   - H1: 36px/44px
   - H2: 30px/38px
   - H3: 24px/32px
   - Body: 16px/24px
   - Small: 14px/20px

3. **Font Weights**
   - Regular: 400
   - Medium: 500
   - Semibold: 600
   - Bold: 700

### 1.2 Color System
1. **Primary Colors**
   - Brand: #0070F3
   - Success: #0070F3
   - Warning: #F5A623
   - Error: #FF0000

2. **Neutral Colors**
   - Background: #FFFFFF
   - Surface: #F5F5F5
   - Text: #171717
   - Border: #E5E5E5

3. **Dark Mode Colors**
   - Background: #0A0A0A
   - Surface: #171717
   - Text: #EDEDED
   - Border: #333333
